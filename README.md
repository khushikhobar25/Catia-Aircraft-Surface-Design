## Aeroplane Surface Modeling using CATIA V5

---
This project demonstrates the design of an aeroplane using **surface modeling techniques** in CATIA V5. The model is developed using the **Generative Shape Design (GSD)** workbench, focusing on smooth aerodynamic surfaces and geometric continuity.
---
### Objective
* Design an aeroplane geometry using surface modeling
* Create a wireframe structure using splines and guide curves
* Generate smooth, continuous surfaces suitable for aerodynamic applications
---
### Tools & Software
* CATIA V5
* Generative Shape Design (GSD)
---
### Methodology
1. **Reference Geometry Creation**
    * Used aircraft sketches/drawings to define base curves
2. **Wireframe Development**
    * Created splines and guide curves
    * Defined cross-sections for fuselage and wings
3. **Surface Modeling**
    * Used multi-section surface (loft) for fuselage
    * Applied sweep operations for wing creation
4. **Surface Continuity**
    * Ensured smooth transitions between surfaces
    * Maintained geometric continuity for aerodynamic shape


### 🔹 Key Skills Demonstrated
* Surface modeling using CATIA (GSD)
* Wireframe creation (splines, guide curves)
* Loft and sweep surface operations
* Aerodynamic shape development
---
### 🔹 Outcome

* Successfully created a smooth aeroplane surface model
* Achieved visually continuous and clean geometry
* Developed practical skills in advanced CAD surfacing
